## 抽獎與裝備分佈前提

### 抽獎盤面裝備格

| 獎品 | 數量 |
|---|---:|
| 0級裝備 | 18 |
| 1級裝備 | 8 |
| 2級裝備 | 5 |
| 3級裝備 | 3 |
| 4級裝備 | 2 |

每盤共 64 格，裝備格共 36 格。  
本表使用簡化平均分佈，不含 0 級裝備合成。

### 裝備加成算法

```
5 類裝備：鎬子、帽子、衣服、褲子、鞋子
每類 10 款式
每個款式取目前持有最高等級
該類總加成 = 10 款式最高等級加總
```

## 50 / 100 抽常用近似值

為了快速做設計估算，可先用：

| 抽數 | 鎬子 | 帽子 | 衣服 | 褲子 | 鞋子 |
|---:|---:|---:|---:|---:|---:|
| 50 | +5 | +5 | +5 | +5 | +5 |
| 100 | +9 | +9 | +9 | +9 | +9 |

### 對能力的意義

| 裝備 | 影響能力 |
|---|---|
| 鎬子 | 挖掘能力 |
| 帽子 | 靈動能力 |
| 衣服 | 背負能力 |
| 褲子 | 修復能力 |
| 鞋子 | 移動速度 |

### 注意

- 這是平均值，單一玩家實際抽獎可能偏科。
- 目前未計算 0 級裝備合成，因此實際長期加成可能略高。
- 150 / 200 抽目前是估算值，進入多人長期模擬前建議再做一次精算。

## 礦山泥土平均產出

> 用途：Must Solve 4 之後反覆測「土方可轉換成多少有效建築長度」時直接引用。

### 基礎挖礦量

目前粗模擬沿用：

```md
每名裸裝玩家每白天約挖 11.4 塊
2 人每白天約挖 22.8 塊
```

| 路線 | 泥土比例 | 2 人每關平均泥土 |

> 輸出位置：`Docs/simulation/monster-drop-sim-log.md`

## 背景

### 經濟基準（來自 economy-sim-log.md）
```
正常玩家日銀幣（純抽獎盤）= 17,862
正常玩家每天通關數 = 120 關
裝備合成總費用 = 188,000 銀/件
```

### 新機制
每隻怪死亡後掉固定銀幣，金額視怪物種類而定。
多人規則：N 人 → 怪物 ×N → 掉落 ×N → ÷N = 與單人等值 → 只模擬單人。

## 計算方式

對每個候選組合：

### 1. 每關銀幣（第 1-20 關）
```
stage_silver(i) = 平民count*平民drop + 跑者count*跑者drop + 猛男count*猛男drop
                + 盾兵count*盾兵drop + 工兵count*工兵drop + Boss count*Boss drop
```

### 2. 正常玩家日收入估算
假設正常玩家（120 關/天）在不同進度農不同關卡：
- **新手期**：農第 5 關（120 次 × stage_silver[5]）
- **中期**：農第 10 關（120 次 × stage_silver[10]）
- **後期**：農第 15 關（120 次 × stage_silver[15]）

每個情境：
```
日怪物銀幣 = 120 × stage_silver(該關)
新日銀幣 = 17,862 + 日怪物銀幣
單件裝備合成天數 = 188,000 ÷ (新日銀幣 × 0.5)
```

### 3. 怪物銀幣佔比
```
怪物銀幣佔比 = 日怪物銀幣 / 新日銀幣 × 100%
```

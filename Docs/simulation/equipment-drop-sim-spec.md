> 建立：2026-06-29
> 目的：測試「怪物掉落銀幣」對進度曲線的影響，並設計合成費用逐級曲線
> 輸出位置：`Docs/simulation/equipment-drop-sim-log.md`

## 背景（Codex 必讀）

### 已定案的基準數值（來自 economy-sim-log.md）

```
正常玩家：120 關/天 → 日銀幣 17,862（僅抽獎盤來源）
極限玩家：600 關/天 → 日銀幣 89,309（僅抽獎盤來源）
裝備合成：單件 Lv0→10 總費用 = 188,000 銀幣
技能點：全 6 屬性滿級 = 34,200 銀幣
```

### 新設計：怪物掉落

- 每關固定掉落**銀幣**（測試多組數值）
- 每關隨機掉落**金幣**（另外測試）
- 多人規則：N 人 → 怪物 ×N → 掉落總量 ×N → 每人 ÷N = 與單人等值
  → 因此只需模擬單人情境

## 任務 1：怪物掉落銀幣影響矩陣

### 測試數值
```js
const MONSTER_SILVER_PER_STAGE = [0, 20, 50, 100, 150, 200, 300, 500];
// 0 = 現況基準（純抽獎盤）
```

### 計算方式

```
日怪物銀幣（正常）= 120 × monster_silver
日怪物銀幣（極限）= 600 × monster_silver

新日銀幣（正常）= 17,862 + 日怪物銀幣
新日銀幣（極限）= 89,309 + 日怪物銀幣

合成預算（正常，50% 分配）= 新日銀幣 × 0.5
單件完成天數（正常）= 188,000 ÷ 合成預算
50 件全滿天數（極限，無分配限制）= 50 × 188,000 ÷ 新日銀幣
```

### 輸出格式（Markdown 表格）

```
| 怪物銀幣/關 | 正常日銀幣 | 正常單件天數 | 極限50件天數 | 是否合理 |
```

**合理判斷標準**（同時滿足）：
- 正常玩家單件天數：10 ～ 30 天（< 10 太快，> 30 太慢）
- 極限玩家 50 件天數：60 ～ 150 天

## 任務 2：合成費用逐級曲線設計

### 限制條件
```
總費用 = 188,000 銀幣（Lv0→10，共 10 次合成）
曲線型態：前低後高（早期便宜、後期貴），讓新手不會棄坑
```

### 設計三個變體

#### 變體 A：線性（基準對照組）
每級固定 = 188,000 ÷ 10 = 18,800

#### 變體 B：二次方曲線（平滑前低後高）
```
cost(n) = a × n²，n = 1..10
sum(1² + 2² + ... + 10²) = 385
a = 188,000 / 385 ≈ 488.3
→ cost(n) = 488.3 × n²（逐項取整後補差值讓總和精確 = 188,000）
```

#### 變體 C：指數曲線（每級 ×1.5）
```
cost(1) = a，cost(n) = a × 1.5^(n-1)
sum = a × (1.5^10 - 1) / (1.5 - 1) ≈ a × 113.33
a = 188,000 / 113.33 ≈ 1,659
→ 逐項取整後補差值讓總和精確 = 188,000
```

#### 變體 D：仿技能曲線（大幅前低後高）
```
前 3 級比例很低（合計 ≈ 10%），第 4 級起每級成長 ≈ 18%
根據此比例分配 188,000
→ 自行計算各級數值，讓總和精確 = 188,000
```

### 每個變體輸出

```
| 升級 | 費用 | 佔總費用% | 正常玩家需攢幾天 |
| Lv0→1 | ... | ...% | ... 天 |
...（共 10 行）
| 合計 | 188,000 | 100% | — |

備註：
- 首次合成（Lv0→1）對應的攢錢天數 = 費用 ÷ (日銀幣 × 0.5)
- 需攢幾天 = 小數點一位
```

## 任務 3：怪物掉落金幣影響（快速評估）

### 測試數值
```js
const MONSTER_GOLD_PER_STAGE_CANDIDATES = [
  { label: '固定5金', fixed: 5, random: 0 },
  { label: '固定10金', fixed: 10, random: 0 },
  { label: '隨機0~20金（平均10）', fixed: 0, random: 20 }, // Math.random() * 21
```

### 計算
```
日怪物金幣（正常）= 120 × 各候選平均金幣/關
新日金幣（正常）= 1,612 + 日怪物金幣

商店每日消費能力（現況金幣 1,612/日）：
  → 銀幣包大 (440金) 每幾天買一次？
新的每幾天？
```

### 輸出格式
```
| 候選 | 正常日金幣 | 銀幣包大購買間隔（天） |
```

遊戲主要玩法&背景
哥布林的信仰
畫面是2D平面類似Terraria

哥布林是主角，人族是敵人
由於哥布林開采了深山太多資源，所以他們憤怒 仇恨 貪婪 想把你裝飾好的家園佔為己有。
遊戲機制 1~4人塔防類遊戲。
把資源裝飾自己的家圍住核心建築

每個哥布林有 (升級會有技能點可以點)
挖掘能力 基礎值=10 (每秒可以敲5次)
疲勞值 = 60
靈動能力 = 50 
背負能力 = 50
修復能力 = 50
移動速度 = 50

霸體不會被攻擊。

裝備圖鑒系統總共10個等級10種類型
稿子 （挖掘能力）
帽子 （靈動能力）
衣服 （背負能力）
褲子 （修復能力）
鞋子 （移動能力）

抽獎系統，每次通關遊戲出來可以獲得票抽獎

哥布林的信仰（核心）
5攻擊力 1秒攻擊頻率1次 50血量 0防禦力 攻擊范圍50
攻速會顯示格式是 1秒攻擊n次

防禦力公式 N/100+N = 100防御力 50%減傷

挖掘能力影響挖礦的速度
沙土石 有50耐久 重量5 10挖掘能力，5下就可以出塊
鐵有100耐久重量10
金有150耐久 重量10
琉璃有500耐久 重量20
鑽有1000耐久 重量50

每個物質的屬性
沙塊 增加 攻擊範圍 1
土塊 增加 血量 1
石塊 增加 防禦0.1
鐵塊 增加 增加塔的物理攻擊 0.1
金塊 增加 增加塔的攻速 0.01次
琉璃 增加 0.1%的魔法傷害(穿透防御力)
鑽塊 增加 連鎖攻擊 0.1人

放置時泥土必須放第一層背景層且必須連續不斷，然後其他物質可以放在第二層有阻隔效果。
泥土蓋到的位置會被當做是核心的Hitbox, 人類可以直接攻擊他扣血。

每10個晚上 出現王，打死之後會出現三張卡片
房主和玩家可以投票倒數10秒，不需要確定 中途可以換票 也可以 不投票
房主強制需要投票，如果沒投會強製選擇第一個選項，房員可以放棄投票，不點或者點了之後再點取消
房主的票數=1.5票

卡片的內容主要分3類
資源型（給物質，立刻獲得 N 沙 土 石 這樣）
能力型（給玩家屬性，本局生效）
增強型（給塔的屬性，本劇生效）

疲勞值
- 每隻哥布林每個白天晚上會恢復60次 最多儲存120次 

靈動能力
- 靠近核心的時候，核心攻擊力和攻速都會獲得巨幅增加
每100點 增加 10%攻擊力和攻速
單人模式的時候額外+15%固定

修復能力
- 可以修復塔的血量
數值/60=每秒回復血量

移動速度
- 在地圖的移動速度
地圖每格是10px, 50ms 玩家每秒可以走5格

背負能力
- 承重能力可以帶多少重量的資源

# WavePlan：關卡波次與多人倍率基準
> 狀態：MVP 定案 / Must Solve 4 已收尾
> 最後更新：2026-06-22
> 用途：集中管理 1-30 關波次、多人倍率、Boss 規則、建造範圍與怪物生成安全區。

---

## 目前問題

我們現在要解的是：Must Solve 4：難度曲線與 1-30 關節奏。

這個問題卡住的是：

- 多人房玩家會同時增加挖礦、建塔、修復與靈動加成。
- 只增加玩家能力、不增加怪物難度，會讓 2-4 人房過早失去壓力。
- 建築地基可以往外延伸，若沒有最大建造範圍，怪物生成點可能被推出地圖或礦山區。
- Boss 關在多人房需要簡單明確的完成條件。

如果不解決，會造成：

- 2 人房測試基準不準。
- 第 20 關 Boss 門檻定位不清楚。
- 玩家可能用超長地基破壞出怪邏輯。
- Claude / Codex 後續模擬與實作會引用不同規則。

## 我的建議

先採用以下 MVP 規則：

```md
N 人房普通怪數量 = 單人基準數量 x N
N 人房 Boss 數量 = N 隻 Boss
全部 Boss 擊殺後，才進入卡片選擇
```

原因：

- 規則直覺，玩家容易理解。
- 實作判斷簡單。
- 多人挖礦與建塔有 1+1>2 的成長傾向，怪物數量 xN 是合理起點。
- 如果後續模擬發現太簡單，再提高特殊怪比例或 Boss 數值。

## 能不能用模擬解決？

可以，但本文件先記錄規則，不在本輪直接模擬。

後續應模擬：

- 單人 / 2 人裸裝是否在第 10 關前死亡。
- 3 / 4 人裸裝是否在第 15 關前死亡。
- 50 抽是否能摸到第 10-15 關區間，但不穩定越過第 20 關。
- 100 抽 + 第 10 關卡片是否能挑戰第 15-20 關，但不穩定越過第 20 關。
- 第 20 關 Boss 是否仍能作為 MVP 硬門檻。
- 第 21-30 關阻擋區是否能阻止玩家穩定通過。

### 敵人成長規則（MVP 定案）

> 這裡是目前開工與模擬的唯一基準。舊的 A/B/C/D 候選測試只保留在 `Docs/simulation/simulation-log-2.md` 作歷史紀錄。

普通怪血量成長：

```md
第 1-10 關：不成長。
第 11-20 關：基礎血量 x 1.05^(關卡 - 11)。
第 21-30 關：使用 MVP 阻擋區規則，每關額外隨機 +15% 到 +25%，逐關累乘。
Boss 不吃普通怪血量成長；Boss 使用自己的獨立數值與 Boss 關特殊規則。
```

普通怪攻擊成長：

```md
第 1-10 關：不成長。
第 11-15 關：跑者 / 工兵攻擊 +1。
第 16-20 關：跑者 / 工兵攻擊 +2。
第 21-30 關：使用 MVP 阻擋區規則，每關額外隨機 +10% 到 +20%，逐關累乘；最低每 2 關至少等效 +1 攻擊壓力。
```

4 人房 Debuff：

```md
4 人房普通怪血量額外 +20%
Boss 不吃這個普通怪 Debuff；Boss20 另有多人門檻規則。
```

Boss 體型與攻擊距離更新：

```md
Boss 身高：4 格
Boss10 攻擊距離：1 格，最高可攻擊高度 5 格
Boss20 起攻擊距離：2 格，最高可攻擊高度 6 格
```

Boss 門口攻擊規則：

```md
若 Boss 關時玩家建築高度 >= 5 格，Boss 可從門口直接攻擊核心 Hitbox。
此攻擊不吃完整建築預射距離。
第 10 Boss 暫不套用完整門口攻擊，避免 50 抽拿不到卡片。
第 20 Boss 起套用門口攻擊。
```

夜晚加時賽 / 狂暴模式：

```md
引用 Docs/game-design-plan.md 既有規則。
晚上 60 秒結束時，如果怪物還沒被清光，進入 30 秒加時賽。
加時賽期間，怪物攻擊力每 5 秒翻一倍。
5 秒 = x2
10 秒 = x4
15 秒 = x8
20 秒 = x16
25 秒 = x32
30 秒 = x64
```

> 此規則同時適用普通關與 Boss 關。Must Solve 4 之後的模擬不得再使用「45 秒後每 5 秒 +10%」的臨時規則。
> 加時賽 30 秒結束時仍未清完怪 → 強制 GameOver（核心判定陣亡，直接進入結算）。

第 20 Boss 多人門檻：

```md
3 人房 Boss20 血量 +30%
4 人房 Boss20 血量 +50%
Boss20 攻擊 6
```

> 這是 MVP 版第 20 關門檻。若正式實玩仍被 3/4 人 100 抽穩定打穿，再優先加 Boss20 機制型壓力，例如遠程攻擊、固定核心壓力、或多人技能頻率加成。

### 第 21-30 關（MVP 暫時阻擋區）

> MVP 目標：第 21-30 關暫時不是精緻可通關曲線，而是用來阻止玩家穩定越過第 20 關後一路通過。正式 21-30 節奏留到 MVP 實玩與卡池擴充後再細修。

第 21-30 關暫用規則：

```md
沿用下方單人基準波次。
N 人房仍套用普通怪數量 x N、Boss x N。
第 21 關起，普通怪每關額外隨機增加血量與攻擊。
血量：每關隨機 +15% 到 +25%，逐關累乘。
攻擊：每關隨機 +10% 到 +20%，逐關累乘；最低每 2 關至少等效 +1 攻擊壓力。
Boss30 使用獨立 Boss 數值，但也可吃 MVP 阻擋用的血量 / 攻擊增壓。
測試與實作時應使用 seed 固定隨機序列，避免每次模擬結果不可重現。
```

設計意圖：

- 第 20 關是目前 MVP 的主要門檻。
- 21-30 暫時不追求公平通關，只防止 50 / 100 抽或強卡直接打穿。
- 之後做 Must Solve 5 卡池與流派後，再回頭把 21-30 做成正式中後期曲線。

## 多人波次倍率

### 普通關

```md
單人波次 = 基準表
N 人房波次 = 每種普通怪數量 x N
```

> 注意：建築長度應視為怪物接觸核心前的預射距離，不可再使用「建築外擴 = 怪物更早攻擊核心」的舊假設。資源可轉換成的有效建築長度已反映在分段建造限制中。

範例：

```md
單人：平民 x8 + 猛男 x5 + 跑者 x4 + 盾兵 x3 + 工兵 x1
2 人：平民 x16 + 猛男 x10 + 跑者 x8 + 盾兵 x6 + 工兵 x2
3 人：平民 x24 + 猛男 x15 + 跑者 x12 + 盾兵 x9 + 工兵 x3
4 人：平民 x32 + 猛男 x20 + 跑者 x16 + 盾兵 x12 + 工兵 x4
```

### 王關

```md
N 人房王關 = 出 N 隻 Boss
全部 Boss 擊殺後，才算王關完成
王關完成後，才進入卡片投票
```

初版 Boss 規則：

- Boss 數量跟玩家數相同。
- Boss 不吃普通怪數量倍率，也不吃 4 人普通怪血量 +20% Debuff。
- Boss20 有獨立多人門檻：3 人血量 +30%、4 人血量 +50%、攻擊 6。
- 不做「1 隻超大 Boss」版本，避免血量倍率與數量倍率同時混在一起。
- 如果畫面壓力過高，後續優先調整 Boss 出生位置、碰撞或攻擊節奏，而不是改回單 Boss。

## 建造範圍與生成安全區

### 地圖基準

```md
地圖寬度：160 格
地圖高度：100 格
核心位置：地圖中心
```

### 水平空間預留

```md
左右兩側各保留礦山安全區：30 格
左右兩側各保留生怪距離：10 格
左右各預留總計：40 格
額外特殊保留：10 格
```

計算：

```md
160 - 40*2 - 10 = 70
70 / 2 = 35
```

### 初版建造限制

```md
核心大小：2x2
核心地基最遠水平距離：分段開放
高度限制：最多 100 格
```

規則：

- 第 0-10 關：左右最多約 14 格
- 第 11-20 關：左右最多約 20 格
- 第 21-30 關：左右最多約 27 格
- 第 30 關後：左右最多約 35 格
- 高度超出 100 格禁止放置。
- 這個限制針對核心地基與連通泥土，避免玩家把有效建築路徑拉到地圖邊界。
- 建築有效長度是怪物接觸核心前的預射距離，不應在模擬中直接視為「怪物更早能攻擊核心」。
- 左右寬度必須低於往上高度；高度需求 = 水平長度 x2。
- 34 格神級玩法成本 = 左 34 + 右 34 + 高 68 = 136 土。
- 粗估土方成本：有效水平長度 x4。
- 後續若加入非核心裝飾物，可另行決定是否受同樣限制。

### 怪物生成安全規則

正常生成：

```md
怪物生成在目前核心 Hitbox / 攻擊覆蓋外 10-20 格。
```

如果空間不足：

```md
怪物生成距離可壓縮為 Hitbox 外 0-10 格。
```

硬性限制：

- 怪物不能生成在礦山範圍內。
- 怪物不能生成在地圖外。
- 怪物不能因為沒有合法生成點而整波失效。
- 若某方向沒有合法點，應改用另一側或上/下方合法生成點。

## 目前 15-20 關基準

### MVP 測試定位

```md
單人 / 2 人裸裝：第 10 關前必須輸。
3 / 4 人裸裝：第 15 關前必須輸。
50 抽：應能摸到第 10-15 關區間，但不穩定越過第 20 關。
100 抽 + 第 10 關卡片：應能挑戰第 15-20 關，但不應穩定越過第 20 關。
第 20 關：MVP 主要硬門檻。
第 21-30 關：暫作阻擋區，不做公平通關承諾。
```

第 19 關單人基準暫定：

```md
平民 x8 + 猛男 x5 + 跑者 x4 + 盾兵 x3 + 工兵 x1
```

2 人房套用 x2 後：

```md
平民 x16 + 猛男 x10 + 跑者 x8 + 盾兵 x6 + 工兵 x2
```

第 20 關：

```md
2 人房出 2 隻 Boss
全部 Boss 擊殺後才選卡
```

第 20 關已定為 MVP 版裝備 / 卡片 / 操作門檻；若實玩中被穩定打穿，優先調 Boss20 或 16-20 壓力，不直接改 1-10 新手段。
